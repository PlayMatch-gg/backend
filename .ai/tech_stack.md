# Технологический стек и утилиты

## Основные технологии

*   **Язык:** Go
*   **Веб-фреймворк:** [Gin](https://github.com/gin-gonic/gin) — быстрый и минималистичный фреймворк для создания REST API.
*   **База данных:** PostgreSQL — реляционная база данных.
*   **ORM:** [GORM](https://gorm.io/) — для взаимодействия с базой данных.
*   **Аутентификация:** JWT (JSON Web Tokens) с использованием библиотеки [golang-jwt/jwt](https://github.com/golang-jwt/jwt).
*   **Хэширование паролей:** `bcrypt` из стандартной библиотеки `golang.org/x/crypto`.
*   **Конфигурация:** [Viper](https://github.com/spf13/viper) — для управления конфигурацией через переменные окружения и `.env` файлы.
*   **Документация API:** [Swag](https://github.com/swaggo/swag) — для генерации Swagger/OpenAPI документации из комментариев в коде.

## Окружение

*   **База данных:** Запускается в Docker-контейнере. Управляется через `docker-compose.yml`.
*   **Сборка и запуск:** Для удобства используется `Makefile` с набором команд для разработки.

## Ключевые команды (`Makefile`)

*   `make db-up`: Запустить контейнер с PostgreSQL.
*   `make dev`: Сгенерировать Swagger-документацию и запустить сервер.
*   `make swag-gen`: Только сгенерировать документацию.
*   `make db-down`: Остановить и удалить контейнер с БД.
