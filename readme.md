# Видение проекта

## Общая идея  
Сервис предназначен для поиска друзей и напарников для кооперативных игр.  
В отличие от внутриигровых лобби, пользователь не привязан ни к конкретной игре, ни к случайному подбору.

---

## Основные элементы

### 1. Профиль пользователя  
- У каждого пользователя есть персональная страница.  
- На странице отображаются игры, которые ему нравятся.  
- Есть система «друзей» — возможность устанавливать коннекты между людьми.  
- Реализована репутационная система:
  - Другие пользователи могут оценивать навыки человека по различным шкалам.  
  - Оценки могут быть как «хардовыми» (зависимыми от жанра), так и «софт-скиллами» (характер и стиль общения).  
- Возможна привязка профиля Steam.

---

### 2. Создание лобби  
- Пользователь может создать собственное лобби.  
- В параметрах указывается:
  - Одна или несколько игр, по которым ищутся сокомандники.  
  - Требуемое количество участников.  
- В будущем может появиться фильтр по рейтингам игроков.  
- После создания лобби открывается групповой чат.  
- Участники подключаются, общаются, знакомятся и ждут заполнения лобби.  
- После этого переходят в игру и играют.  
- В перспективе возможно добавление голосового чата.

---

### 3. Главная страница  
- Главная страница — список открытых лобби.  
- Присоединение к существующим лобби должно приносить репутацию, чтобы мотивировать пользователей заходить.  
- Доступны:
  - Фильтры по играм.  
  - Сортировки (по рейтингу хоста, по времени — от старых к новым).  
- Сформированные (полные) лобби скрываются.

---

### 4. Добавление игр  
- Ручное добавление игр неудобно — нужен механизм подсказок.  
- Пользователь отправляет ссылку на игру в Steam.  
- Сервис парсит данные и добавляет игру в базу:
  - Картинка  
  - Название  
  - Описание  
- Через админку можно будет добавлять теги.

---

### 5. Монетизация  
- Потенциальная монетизация может происходить через продвижение отдельных игр.  
- Однако проект в первую очередь идейный.
